Parte 1

Che cos'è una struct

Una struct permette di raggruppare dati diversi sotto un unico nome. Se un prodotto ha id, nome, prezzo e quantita, invece di gestire quattro variabili separate puoi creare un tipo unico.

Definire un tipo

Con typedef struct crei un nuovo tipo riutilizzabile nel programma.

#define MAXSTR 31

typedef struct {
    int id;
    char nome[MAXSTR];
    float prezzo;
    int quantita;
} TProdotto;

Usare una variabile

Una variabile struct contiene tutti i suoi campi. Con il punto accedi a un campo specifico.

TProdotto prodotto;

prodotto.id = 10;
prodotto.prezzo = 4.50f;
prodotto.quantita = 20;
strcpy_s(prodotto.nome, MAXSTR, "quaderno");

La struct è un modello

La definizione dice quali campi avra ogni variabile di quel tipo. Le variabili vere sono quelle dichiarate dopo: TProdotto p1;, TProdotto p2;, oppure TProdotto magazzino[100];.

Campo Esempio Quando serve
int id 101 Codice numerico, posizione logica, matricola.
char nome[MAXSTR] "penna" Testi brevi salvati dentro la struct.
float prezzo 1.25f Numeri con virgola, come prezzi o medie.
int quantita 12 Conteggi interi.

Studente

typedef struct {
    int matricola;
    char nome[40];
    float media;
} TStudente;

Libro

typedef struct {
    char titolo[60];
    char autore[40];
    int pagine;
} TLibro;

Punto 2D

typedef struct {
    float x;
    float y;
} TPunto;
Parte 2

Array di struct

Un array di struct contiene tanti elementi dello stesso tipo. Di solito si usa una costante per la capienza massima e una variabile n per dire quanti elementi sono davvero validi. 

#define MAX_PROD 100

TProdotto prodotti[MAX_PROD] = {
    {1, "penna", 1.20f, 50},
    {2, "quaderno", 3.40f, 25},
    {3, "gomma", 0.70f, 80}
};

int n = 3;
1

Capienza massima

MAX_PROD dice quanto spazio hai riservato.

2

Elementi validi

n dice quante posizioni contengono dati utili.

3

Cicli

Per stampare o cercare usi i < n, non i < MAX_PROD.

Stampare un array

void stampaProdotti(TProdotto ar[], int n)
{
    int i;

    for (i = 0; i < n; i++)
    {
        printf("%d %s %.2f %d\n",
            ar[i].id,
            ar[i].nome,
            ar[i].prezzo,
            ar[i].quantita);
    }
}

Cercare in un array

int cercaPerId(TProdotto ar[], int n, int id)
{
    int i;

    for (i = 0; i < n; i++)
    {
        if (ar[i].id == id)
        {
            return i;
        }
    }

    return -1;
}
Parte 3

Funzioni ordinate

Una funzione dovrebbe fare un lavoro preciso: stampare, cercare, modificare, aggiungere, eliminare, ordinare. Il prototipo dice che dati riceve e che risultato restituisce.

Solo lettura

Se la funzione deve solo leggere, passa i valori normalmente.

void stampaProdotti(
    TProdotto ar[],
    int n
);

Modifica dati

Per modificare elementi dell'array basta passare l'array: arriva gia come indirizzo.

int modificaPrezzo(
    TProdotto ar[],
    int n,
    int id,
    float prezzo
);

Modifica n

Se cambia il numero di elementi, la funzione deve ricevere un puntatore a n.

int aggiungiProdotto(
    TProdotto ar[],
    int* pN,
    TProdotto prod
);

Perché spesso si restituisce int

In molti esercizi si usa 1 per dire "operazione riuscita" e 0 per dire "operazione non riuscita". Se devi cercare una posizione, invece, spesso restituisci l'indice oppure -1. 

if (aggiungiProdotto(prodotti, &n, nuovo))
{
    printf("Prodotto aggiunto.\n");
}
else
{
    printf("Array pieno.\n");
}
Parte 4

Puntatori, & e *

Un puntatore è una variabile che contiene un indirizzo. Serve quando vuoi lavorare sul dato originale invece che su una copia.

Variabile

int n = 3;

La variabile contiene il valore.

Indirizzo

&n

L'operatore & prende l'indirizzo.

Valore puntato

*pN = 4;

L'operatore * usa il valore dentro quell'indirizzo.

Il punto .

Si usa quando hai una struct normale.

TProdotto p;

p.prezzo = 2.50f;
p.quantita = 10;
strcpy_s(p.nome, MAXSTR, "righello");

La freccia ->

Si usa quando hai un puntatore a struct.

void cambiaPrezzo(TProdotto* p)
{
    p->prezzo = 2.50f;
    p->quantita = 10;
}

Quando uso * e &

Situazione Nel prototipo Nella chiamata Motivo
La funzione legge un intero int n stampa(n) Basta una copia.
La funzione modifica un intero int* pN aggiungi(&n) Serve l'indirizzo della variabile originale.
La funzione modifica una struct singola TProdotto* p sconta(&prod) La struct originale deve cambiare.
La funzione lavora su un array TProdotto ar[] stampa(ar, n) Il nome dell'array si comporta gia come indirizzo del primo elemento.
Parte 5

Stringhe dentro le struct

In C una stringa è un array di char. Dopo la dichiarazione non puoi assegnare un array con =: devi copiare i caratteri dentro lo spazio disponibile.

Errore

prodotto.nome = "matita";
prodotto->nome = "matita";

Il campo nome è un array: non si assegna cosi.

Corretto

strcpy_s(prodotto.nome, MAXSTR, "matita");
strcpy_s(prodotto->nome, MAXSTR, "matita");

Con una struct normale usi il punto. Con un puntatore a struct usi la freccia.

Confrontare stringhe

Anche il confronto tra stringhe non si fa con ==. Per confrontare il contenuto si usa strcmp: restituisce 0 quando le stringhe sono uguali. 

if (strcmp(ar[i].nome, nomeCercato) == 0)
{
    printf("Prodotto trovato.\n");
}
Parte 6

Errori comuni

Sono gli inciampi tipici quando si lavora con struct, array, stringhe e puntatori. Conviene imparare a riconoscerli al volo.

Confondere MAX e n

for (i = 0; i < n; i++)
{
    stampa(ar[i]);
}

La capienza massima non è il numero di elementi validi.

Dimenticare &

aggiungiProdotto(ar, &n, nuovo);

Se la funzione riceve int* pN, nella chiamata passi &n.

Usare . al posto di ->

void sconta(TProdotto* p)
{
    p->prezzo = p->prezzo * 0.90f;
}

Con un puntatore a struct si usa la freccia.

Assegnare stringhe con =

strcpy_s(ar[i].nome, MAXSTR, nuovoNome);

I campi char[] si riempiono con funzioni di copia.

Non controllare i limiti

if (*pN >= MAX_PROD)
{
    return 0;
}

Prima di aggiungere controlla che l'array non sia pieno.

Sbagliare la cancellazione

for (i = pos; i < n - 1; i++)
{
    ar[i] = ar[i + 1];
}
n--;

Eliminare da array statico significa spostare a sinistra gli elementi successivi.

File di esempio

Un esempio completo e generale su prodotti: mostra struct, array, ricerca, modifica, aggiunta, cancellazione e ordinamento.

C 
#include <stdio.h>
#include <string.h>

#define MAXSTR 31
#define MAX_PROD 100

typedef struct {
    int id;
    char nome[MAXSTR];
    float prezzo;
    int quantita;
} TProdotto;

void stampaProdotti(TProdotto ar[], int n);
int cercaPerId(TProdotto ar[], int n, int id);
int modificaPrezzo(TProdotto ar[], int n, int id, float nuovoPrezzo);
void applicaSconto(TProdotto* p, float percentuale);
int aggiungiProdotto(TProdotto ar[], int* pN, TProdotto nuovo);
int eliminaInPosizione(TProdotto ar[], int* pN, int posizione);
void ordinaPerPrezzoCrescente(TProdotto ar[], int n);

int main(void)
{
    TProdotto magazzino[MAX_PROD] = {
        {1, "penna", 1.20f, 50},
        {2, "quaderno", 3.40f, 25},
        {3, "gomma", 0.70f, 80}
    };

    int n = 3;
    int pos;
    TProdotto nuovo = {4, "matita", 0.90f, 100};

    stampaProdotti(magazzino, n);

    modificaPrezzo(magazzino, n, 2, 3.10f);

    pos = cercaPerId(magazzino, n, 1);
    if (pos != -1)
    {
        applicaSconto(&magazzino[pos], 10.0f);
    }

    if (aggiungiProdotto(magazzino, &n, nuovo))
    {
        printf("Prodotto aggiunto.\n");
    }

    eliminaInPosizione(magazzino, &n, 2);
    ordinaPerPrezzoCrescente(magazzino, n);
    stampaProdotti(magazzino, n);

    return 0;
}

void stampaProdotti(TProdotto ar[], int n)
{
    int i;

    for (i = 0; i < n; i++)
    {
        printf("%3d %-20s %7.2f %4d\n",
            ar[i].id,
            ar[i].nome,
            ar[i].prezzo,
            ar[i].quantita);
    }
}

int cercaPerId(TProdotto ar[], int n, int id)
{
    int i;

    for (i = 0; i < n; i++)
    {
        if (ar[i].id == id)
        {
            return i;
        }
    }

    return -1;
}

int modificaPrezzo(TProdotto ar[], int n, int id, float nuovoPrezzo)
{
    int pos = cercaPerId(ar, n, id);

    if (pos == -1)
    {
        return 0;
    }

    ar[pos].prezzo = nuovoPrezzo;
    return 1;
}

void applicaSconto(TProdotto* p, float percentuale)
{
    p->prezzo = p->prezzo * (1.0f - percentuale / 100.0f);
}

int aggiungiProdotto(TProdotto ar[], int* pN, TProdotto nuovo)
{
    if (*pN >= MAX_PROD)
    {
        return 0;
    }

    ar[*pN] = nuovo;
    (*pN)++;
    return 1;
}

int eliminaInPosizione(TProdotto ar[], int* pN, int posizione)
{
    int i;
    int n = *pN;

    if (posizione < 0 || posizione >= n)
    {
        return 0;
    }

    for (i = posizione; i < n - 1; i++)
    {
        ar[i] = ar[i + 1];
    }

    (*pN)--;
    return 1;
}

void ordinaPerPrezzoCrescente(TProdotto ar[], int n)
{
    int i;
    int j;
    TProdotto temp;

    for (i = 0; i < n - 1; i++)
    {
        for (j = 0; j < n - 1 - i; j++)
        {
            if (ar[j].prezzo > ar[j + 1].prezzo)
            {
                temp = ar[j];
                ar[j] = ar[j + 1];
                ar[j + 1] = temp;
            }
        }
    }
}
