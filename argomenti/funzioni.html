Funzioni in C

Funzioni, return, parametri.

Una funzione è un blocco di codice con un nome. Può non restituire nulla, può restituire un valore, può leggere parametri e può modificare variabili del main quando riceve il loro indirizzo.

Parte 1

Tipo di ritorno

Il tipo prima del nome della funzione dice cosa la funzione restituisce al punto in cui viene chiamata.

void

La funzione fa qualcosa, ma non produce un valore da salvare.

void stampaNumeriDa1a10()
{
    int i;

    for (i = 1; i <= 10; i++)
    {
        printf("%d ", i);
    }
}

int, char, struct

Se il tipo non è void, serve un return compatibile.

int sePari(int n)
{
    if (n % 2 == 0)
    {
        return 1;
    }

    return 0;
}

Regola secca

Parte 2

Parametri normali

I parametri sono le variabili scritte tra parentesi. Se li passi normalmente, la funzione riceve una copia.

Leggere un valore

Qui n serve solo per decidere se è pari.

int sePari(int n)
{
    return n % 2 == 0;
}

Modifica temporanea

Modificare x qui non cambia la variabile originale nel main.

void prova(int x)
{
    x = 99;
}

int a = 3;
prova(a);   // a resta 3
Parte 3

Parametri puntati

Se vuoi modificare davvero una variabile del main, passi il suo indirizzo con & e nella funzione usi un puntatore.

Intero puntato

void aumenta(int* pN)
{
    (*pN)++;
}

int n = 3;
aumenta(&n);  // n diventa 4

&n prende l'indirizzo. *pN prende il valore dentro quell'indirizzo.

Struct puntata

void sconto(TProdotto* prodotto)
{
    int sconto;

    scanf("%d", &sconto);
    prodotto->prezzo =
        prodotto->prezzo * (100 - sconto) / 100;
}

Parte 4

Array come parametri

Gli array sono speciali: quando li passi a una funzione non copi tutto l'array. La funzione lavora già sullo spazio originale.

Array di interi

int sommaArray(int ar[], int n)
{
    int i;
    int somma = 0;

    for (i = 0; i < n; i++)
    {
        somma += ar[i];
    }

    return somma;
}

Array di struct

int aggiungiProdotto(TProdotto ar[], int* pN, TProdotto nuovo)
{
    if (*pN >= MAX_PROD)
    {
        return 0;
    }

    ar[*pN] = nuovo;
    (*pN)++;
    return 1;
}
Codice

File di esempio

Un esempio compatto con tipi di ritorno, parametri normali e parametri puntati.

C 
#include <stdio.h>

void stampaNumeriDa1a10(void);
int sePari(int n);
void aumenta(int* pN);

int main(void)
{
    int numero = 4;
    int esito;

    stampaNumeriDa1a10();

    esito = sePari(numero);
    if (esito)
    {
        printf("Pari\n");
    }
    else
    {
        printf("Dispari\n");
    }

    aumenta(&numero);
    printf("Ora numero vale %d\n", numero);

    return 0;
}

void stampaNumeriDa1a10(void)
{
    int i;

    for (i = 1; i <= 10; i++)
    {
        printf("%d ", i);
    }
}

int sePari(int n)
{
    return n % 2 == 0;
}

void aumenta(int* pN)
{
    (*pN)++;
}
