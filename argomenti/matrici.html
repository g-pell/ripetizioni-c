Parte 1

Che cos'è una matrice

Una matrice è un array a due dimensioni. Invece di avere solo una posizione, ogni elemento ha una riga e una colonna.

Dichiarazione

#define RIGHE 3
#define COLONNE 3

int matrice[RIGHE][COLONNE] = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

Coordinate

Gli indici partono da 0. In una 3x3 le righe sono 0, 1, 2 e le colonne sono 0, 1, 2.

Espressione Significato
matrice[0][0] Prima riga, prima colonna.
matrice[1][2] Seconda riga, terza colonna.
matrice[i][j] Cella alla riga i e colonna j.
Parte 2

Come leggere una cella

Pensa a battaglia navale: scegli una riga, scegli una colonna, e prendi la cella dove si incrociano.

Esempio: matrice[1][2]

Riga 1 è {4, 5, 6}. Colonna 2 è la terza colonna. L'incrocio è 6. 

printf("%d", matrice[1][2]);  // stampa 6

matrice[1][2] = 20;           // modifica quella cella
Parte 3

Cicli annidati

Il ciclo esterno scorre le righe. Per ogni riga, il ciclo interno scorre tutte le colonne.

Schema standard

int i;
int j;

for (i = 0; i < RIGHE; i++)
{
    for (j = 0; j < COLONNE; j++)
    {
        printf("%d ", matrice[i][j]);
    }

    printf("\n");
}

Cosa succede

Prima leggi tutta la riga 0, poi tutta la riga 1, poi tutta la riga 2.

i = 0: 1 2 3
i = 1: 4 5 6
i = 2: 7 8 9
Parte 4

Funzioni con matrici

Quando passi una matrice a una funzione, devi specificare almeno il numero di colonne. Non serve scrivere * nella chiamata: la matrice arriva già come indirizzo.

Somma di tutti i valori

int sommaMatrice(int matrice[][COLONNE])
{
    int i;
    int j;
    int somma = 0;

    for (i = 0; i < RIGHE; i++)
    {
        for (j = 0; j < COLONNE; j++)
        {
            somma += matrice[i][j];
        }
    }

    return somma;
}

Conta occorrenze

int cercaOccorrenze(int matrice[][COLONNE], int valore)
{
    int i;
    int j;
    int contatore = 0;

    for (i = 0; i < RIGHE; i++)
    {
        for (j = 0; j < COLONNE; j++)
        {
            if (matrice[i][j] == valore)
            {
                contatore++;
            }
        }
    }

    return contatore;
}

Matrice di output

Se vuoi creare una seconda matrice con il risultato, la passi come parametro e la riempi dentro la funzione. 

void pariDispariMatrice(int matrice[][COLONNE], char risultato[][COLONNE])
{
    int i;
    int j;

    for (i = 0; i < RIGHE; i++)
    {
        for (j = 0; j < COLONNE; j++)
        {
            if (matrice[i][j] % 2 == 0)
            {
                risultato[i][j] = 'P';
            }
            else
            {
                risultato[i][j] = 'D';
            }
        }
    }
}

File di esempio

C 
#include <stdio.h>

#define RIGHE 3
#define COLONNE 3

void stampaMatrice(int matrice[][COLONNE]);
int sommaMatrice(int matrice[][COLONNE]);
int cercaOccorrenze(int matrice[][COLONNE], int valore);
void pariDispariMatrice(int matrice[][COLONNE], char risultato[][COLONNE]);

int main(void)
{
    int matrice[RIGHE][COLONNE] = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };

    char risultato[RIGHE][COLONNE];

    stampaMatrice(matrice);
    printf("Somma: %d\n", sommaMatrice(matrice));
    printf("Occorrenze di 2: %d\n", cercaOccorrenze(matrice, 2));

    pariDispariMatrice(matrice, risultato);

    return 0;
}
