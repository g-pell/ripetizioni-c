Ripasso generale

Clienti, piatti, funzioni.

Questo esercizio mette insieme più struct, array paralleli come contenitori, funzioni con codici di errore, input da tastiera, ricerca con stringhe e modifiche tramite puntatori.

Parte 1

Cosa hai costruito

Il programma simula una piccola gestione di cucina: hai una lista di piatti disponibili e una lista di clienti. Un cliente può comprare un piatto solo se il piatto esiste, costa abbastanza poco per il suo portafoglio e ha quantità disponibile.

Cliente

Ha dati personali e soldi disponibili. Se acquista, il portafoglio dovrebbe diminuire.

Piatto

Ha codice, nome, prezzo, descrizione e quantità. Se viene venduto, la quantità diminuisce.

Esiti

Le funzioni restituiscono codici per distinguere riuscita, soldi insufficienti, piatto mancante o quantità finita.

Parte 2

Struct diverse

Qui non hai una sola struct: hai due tipi diversi perché cliente e piatto rappresentano oggetti diversi. È una scelta pulita: ogni struct contiene solo i dati che le appartengono.

Tipo Campi Perché
TCliente nome, cognome, portafoglio Descrive chi compra.
TPiatto codice, nome, prezzo, descrizione, quantita Descrive cosa si può comprare.
Parte 3

Aggiungere un piatto

Per aggiungere un elemento in fondo a un array devi usare *nPiatti come posizione libera, poi incrementare davvero nPiatti.

Idea corretta

piatti[*nPiatti] = piattoDaAggiungere;
(*nPiatti)++;

Se nPiatti vale 1, la prossima posizione libera è proprio indice 1.

Errore tipico

piatti[(*nPiatti) - 1] = piattoDaAggiungere;
nPiatti++;

Così sovrascrivi l'ultimo elemento valido e incrementi il puntatore, non il valore puntato.

Parte 4

Acquistare un piatto

La funzione cerca il piatto per nome, controlla gli errori possibili e poi modifica quantità e portafoglio. Qui è importante decidere se il cliente deve cambiare davvero nel main.

Codici di errore

Codice Significato Quando succede
0 Operazione riuscita Piatto trovato, soldi sufficienti, quantità disponibile.
-1 Soldi insufficienti Il portafoglio del cliente è minore del prezzo.
-2 Piatto non esistente La ricerca con strcmp non trova il nome.
-3 Quantità finita Il piatto esiste ma quantita <= 0.

Cliente passato per valore

int acquistaPiatto(..., TCliente cliente)
{
    cliente.portafoglio -= prezzo;
}

Modifica solo una copia. Nel main il cliente resta con lo stesso portafoglio.

Cliente passato per puntatore

int acquistaPiatto(..., TCliente* cliente)
{
    cliente->portafoglio -= prezzo;
}

Modifica il cliente originale. La chiamata diventa &clienti[1].

Parte 5

Alternative sane

Lo stesso problema si può risolvere in più modi. L'importante è essere coerenti con cosa vuoi modificare e cosa vuoi solo leggere.

Riempire a mano

TPiatto p = {3, "Pizza", 8.50f, "Margherita", 10};

Comodo per test veloci e dati già noti.

Riempire da tastiera

scanf("%d", &p.codice);
scanf("%29s", p.nome);
scanf("%f", &p.prezzo);

Con %29s eviti di scrivere oltre l'array da 30 char.

Aggiungere molti piatti

for (i = 0; i < nDaAggiungere; i++)
{
    if (!aggiungiPiatto(piatti, &nPiatti, nuovi[i]))
    {
        break;
    }
}

Il ciclo si ferma se l'array è pieno.

Cercare per codice

if (piatti[i].codice == codiceCercato)
{
    return i;
}

Il codice è spesso meglio del nome perché è identificativo e non ambiguo.

Codice

Versione ripulita

Questa versione conserva l'idea dell'esercizio, ma sistema i punti delicati: aggiunta in coda, incremento di nPiatti, cliente modificato davvero e input limitato.

C 
#include <stdio.h>
#include <string.h>

#define MAXCHAR 30
#define MAXCLIENTI 10
#define MAXPIATTI 20

typedef struct {
    char nome[MAXCHAR];
    char cognome[MAXCHAR];
    float portafoglio;
} TCliente;

typedef struct {
    int codice;
    char nome[MAXCHAR];
    float prezzo;
    char descrizione[MAXCHAR];
    int quantita;
} TPiatto;

int aggiungiPiatto(TPiatto piatti[], int* pNPiatti, TPiatto nuovo);
int cercaPiattoPerNome(TPiatto piatti[], int nPiatti, char nomePiatto[]);
int acquistaPiatto(TPiatto piatti[], int nPiatti, char nomePiatto[], TCliente* cliente);

int main(void)
{
    TCliente clienti[MAXCLIENTI] = {
        {"Luca", "Rossi", 150.2f},
        {"Marco", "Verdi", 75.4f}
    };

    TPiatto piatti[MAXPIATTI] = {
        {123, "Carbonara", 15.1f, "Pasta", 5}
    };

    int nPiatti = 1;
    TPiatto nuovo = {456, "Pizza", 8.5f, "Margherita", 10};
    char piattoDaAcquistare[MAXCHAR] = "Pizza";
    int esito;

    if (!aggiungiPiatto(piatti, &nPiatti, nuovo))
    {
        printf("Errore nell'aggiunta del piatto.\n");
        return 1;
    }

    esito = acquistaPiatto(piatti, nPiatti, piattoDaAcquistare, &clienti[1]);

    if (esito == 0)
    {
        printf("Acquisto riuscito.\n");
    }
    else
    {
        printf("Errore acquisto: %d\n", esito);
    }

    return 0;
}

int aggiungiPiatto(TPiatto piatti[], int* pNPiatti, TPiatto nuovo)
{
    if (*pNPiatti >= MAXPIATTI)
    {
        return 0;
    }

    piatti[*pNPiatti] = nuovo;
    (*pNPiatti)++;
    return 1;
}

int cercaPiattoPerNome(TPiatto piatti[], int nPiatti, char nomePiatto[])
{
    int i;

    for (i = 0; i < nPiatti; i++)
    {
        if (strcmp(piatti[i].nome, nomePiatto) == 0)
        {
            return i;
        }
    }

    return -1;
}

int acquistaPiatto(TPiatto piatti[], int nPiatti, char nomePiatto[], TCliente* cliente)
{
    int indice = cercaPiattoPerNome(piatti, nPiatti, nomePiatto);

    if (indice == -1)
    {
        return -2;
    }

    if (cliente->portafoglio < piatti[indice].prezzo)
    {
        return -1;
    }

    if (piatti[indice].quantita <= 0)
    {
        return -3;
    }

    piatti[indice].quantita--;
    cliente->portafoglio -= piatti[indice].prezzo;
    return 0;
}
